### Setup

Install [yarn][yarn]. Then:

```
$ git clone https://github.com/rttss-sahil/conway-game-of-life.git
$ cd conway-game-of-life
$ yarn
$ yarn start
```

Visit http://localhost:3000/ to view your development server.

### Testing

```
$ yarn start
```
